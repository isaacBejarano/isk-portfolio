Culex

Culex.js is an easy-to-use and lightweight ES5 library to fetch JSON data from RESTful APIs through the HTTP protocol by using the ES5 XMLHttpRequest object. Coded in TypeScript, the library is available in both TypeScript and ES5 versions. A minified ES5 version is also provided.
